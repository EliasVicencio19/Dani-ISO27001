## Resumen de la Rama
Esta rama principal (`main`) contiene el proyecto Dani-ISO27001, enfocado en la implementación y gestión de estándares ISO 27001 para la seguridad de la información. El repositorio está actualmente en desarrollo inicial, con el archivo README.md como punto de partida.

Esta rama (`docs`) contiene la documentación completa y oficial del proyecto Dani-ISO27001

- Directrices de Proyecto y Transferencia Tecnológica
- Manual de Usuario Dani Platform v3
- Plan de Desarrollo Dani Platform v2
- Documentos de especificación UX, proyecto, estándares, etc.
- Normativas ISO 27001:2022 (ESP e inglés)
- Documento de problema de negocio
